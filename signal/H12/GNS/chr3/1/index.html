Guinea An. gambiae | H12 | Chromosome 3 | Signal #1¶

This page describes a signal of selection found in the Guinea An. gambiae population using the H12 (Garud et al. 20XX) statistic.The inferred focus of this signal is on chromosome arm 3R between positions 28,480,001 and 28,600,000.

The following 11 genes overlap the focal region: AGAP009184, AGAP009185, AGAP009187 (Indanol dehydrogenase), AGAP009188 (solute carrier family 39 (zinc transporter), member 9), AGAP009189, AGAP009190 (GSTE8 - glutathione S-transferase epsilon class 8), AGAP009191 (GSTE6 - glutathione S-transferase epsilon class 6), AGAP009192 (GSTE5 - glutathione S-transferase epsilon class 5), AGAP009193 (GSTE4 - glutathione S-transferase epsilon class 4), AGAP009194 (GSTE2 - glutathione S-transferase epsilon class 2), AGAP009195 (GSTE1 - glutathione S-transferase epsilon class 1).

The following 6 genes are within 50 kbp of the focal region: AGAP009196 (GSTE7 - glutathione S-transferase epsilon class 7), AGAP009197 (GSTE3 - glutathione S-transferase epsilon class 3), AGAP009198, AGAP009199 (palmitoyltransferase ZDHHC24), AGAP009200 (collagen type IV alpha), AGAP009201 (collagen type IV alpha).
