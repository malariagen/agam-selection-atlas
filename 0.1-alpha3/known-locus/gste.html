Gste locus¶

This page provides information about the Glutathione S-transferase epsilon gene cluster, a locus for which there is prior evidence of recent positive selection and/or an association with insecticide resistance in Anopheles mosquitoes.

The locus occurs on chromosome arm 3R from position 28,591,663 to 28,602,280.

The locus contains the following 8 genes: AGAP009190 (GSTE8 - glutathione S-transferase epsilon class 8), AGAP009191 (GSTE6 - glutathione S-transferase epsilon class 6), AGAP009192 (GSTE5 - glutathione S-transferase epsilon class 5), AGAP009193 (GSTE4 - glutathione S-transferase epsilon class 4), AGAP009194 (GSTE2 - glutathione S-transferase epsilon class 2), AGAP009195 (GSTE1 - glutathione S-transferase epsilon class 1), AGAP009196 (GSTE7 - glutathione S-transferase epsilon class 7), AGAP009197 (GSTE3 - glutathione S-transferase epsilon class 3).

References¶

  • Mitchell et al. (2014) Metabolic and Target-Site Mechanisms Combine to Confer Strong DDT Resistance in Anopheles gambiae
