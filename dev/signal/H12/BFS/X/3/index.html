This page describes a signal of selection found in the Burkina Faso An. gambiae population using the H12 (Garud et al. 20XX) statistic.The focus of this signal is on chromosome arm X between positions 9,760,000 and 9,800,000. The evidence supporting this signal is strong ( >= 100 on both flanks).

Signal location. Blue markers show the values of the selection statistic. The dashed black line shows the fitted peak model. The shaded red area shows the focus of the selection signal.
