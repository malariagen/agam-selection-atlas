Rdl locus
=========




This page provides information about the **GABA-gated chloride channel subunit gene**, a locus for which there is prior
evidence of recent positive selection and/or an association with insecticide resistance in
*Anopheles* mosquitoes.

The locus occurs on chromosome arm **2L**
from position **25,363,652** to **25,434,556**.


The locus contains gene :doc:`../gene/AGAP006028` (Rdl - GABA-gated chloride channel subunit).
