Vgsc locus
==========




This page provides information about the **Voltage-gated sodium channel gene**, a locus for which there is prior
evidence of recent positive selection and/or an association with insecticide resistance in
*Anopheles* mosquitoes.

The locus occurs on chromosome arm **2L**
from position **2,358,158** to **2,431,617**.


The locus contains gene :doc:`../gene/AGAP004707` (para - voltage-gated sodium channel).
